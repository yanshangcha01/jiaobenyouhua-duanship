# 短视频脚本写作模板 v3.0

> 每次写脚本时，按以下步骤执行，不要跳步。

---

## ⚠️ AI 画面负面提示词（脚本全局强制声明）

> **以下负面提示词必须出现在：**
> 1. **脚本开头**的"自检"区域（让阅读者一眼看到）
> 2. **每一条** AI 绘图提示词的开头
> 3. **每一条** AI 视频提示词的开头

```
不要字幕，不要背景音乐，人物要有真实感，保持真实皮肤的质感，食物类画面要贴近真实食物，自然状态下的纹理和质感，避免过度磨皮和塑料感
```

## 附录：提示词质量自检清单（每条 AI 提示词必过）

### AI 绘图提示词 8 要素

```
□ 人物外貌（年龄/性别/发型/肤色/五官）
□ 人物服装（上装/下装/鞋/配饰）
□ 姿势（站/坐/蹲 + 身体朝向）
□ 表情（眼睛/嘴型/眉毛）
□ 场景（地点/环境/品牌元素）
□ 光线（自然光/暖光/侧光/逆光）
□ 构图（9:16/16:9 + 景别 + 角度）
□ 风格调性（真实摄影/浅景深/不要卡通/不要插画）
```

### AI 视频提示词 5 必填 + 3 可选（@ 角色参考图版）

```
【必填项】（5 项，参考图给不出，必须详细写）
□ AI 负面提示词（开头强制）
□ 动作序列（按时间顺序）
□ 台词原文（口型同步）
□ 话语对应动作（每句话对应什么动作）
□ 表情变化（阶段描述）
□ 镜头/光线/时长/比例
□ 场景（地点/环境/品牌元素）

【可省略项】（3 项，@ 角色参考图后 AI 自动读）
- 人物外貌（年龄/性别/发型/肤色/五官）— 用 @角色名 代替
- 人物服装（上装/下装/鞋/配饰）— 用 @角色名 代替
- 气质关键词（亲和/活泼等）— 用 @角色名 代替
```
