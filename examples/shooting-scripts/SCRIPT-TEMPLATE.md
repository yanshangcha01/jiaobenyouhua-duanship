# 短视频分镜脚本模板

> 每次写脚本时，按以下步骤执行，不要跳步。

## Step 1: 台词拆解（必填）

对原始口播文案的每一句话进行拆解：

| 台词原文 | 传递的核心信息 | 情绪基调 | 应匹配的场景 | 人物的动作设计 |
|---------|-------------|---------|------------|-------------|
| （填写原话） | （什么卖点） | （兴奋/惊讶/真诚等） | （地点/产品/环境等） | （手/表情/眼神） |

## Step 2: 角色与环境确认（必填）

- **出镜人物**：姓名/风格/穿着
- **门店名称**：全称
- **拍摄地点**：具体位置
- **角色一致性检查**：本次脚本使用哪个定妆照作为人物参考？

## Step 3: 分镜表格（必填）

| # | 时长 | 景别 | 运镜 | 画面内容 | 台词 | 表情细节 | 姿势 | 手势 | 眼神 | 人物定妆照 | 实拍素材路径 | AI生成需求 |
|---|------|------|------|---------|------|---------|------|------|------|---------|-------------|-----------|
| 1 | | | | | | | | | | | | |

> 表情细节必填项：眼睛（睁大/眯眼/瞪圆）、嘴型（微笑/张大/抿嘴）、眉毛（上扬/紧皱）

## Step 4: 6问自检（必填）

- [ ] Q1：这个脚本写的是什么人？——（回答）
- [ ] Q2：在哪里？——（回答）
- [ ] Q3：在干什么事？——（回答）
- [ ] Q4：观众能看到并理解吗？——（回答）
- [ ] Q5：视频节奏合理吗？——（回答）
- [ ] Q6：每个分镜都有对应的AI提示词吗？——（回答）

## Step 5: AI绘图提示词（必填）

为每一个分镜编写一条Midjourney/Stable Diffusion提示词：

**分镜#** 提示词：
```
（中文描述，包含：人物外貌+服装+姿势+表情+场景+光线+构图+风格）
```

## Step 5.5：视频生成提示词（必填）

为每一个分镜编写一条AI视频生成提示词（用于即梦视频/可灵/Kling/Runway/Sora等）：

| 项目 | 内容 |
|------|------|
| **景别** | （近景/远景/全景/特写/超特写 + 俯拍/仰拍/平拍/正面/侧面等） |
| **运镜** | （固定镜头/缓慢推进/拉远/跟拍/横移/环绕/快切等） |
| **时长** | 几秒～几秒钟 |
| **人物位置** | 人物在哪里（场景描述） |
| **动作序列** | 按时间顺序列出人物做的一系列动作 |
| **台词** | 这句分镜的台词原文 |
| **话语对应的动作** | 说到每句话时人物在做什么动作（逐句对应） |
| **表情** | 整个分镜的表情变化（阶段描述） |
| **说话的语气** | 语速/音调/重音/情感色彩 |
| **人物情绪** | 整体情绪状态 |
| **画面细节** | 光线/背景/道具/氛围等 |
| **视频提示词** | `（一段可以直接复制进AI视频工具的完整中文描述，必须包含台词原文，用于人物对口型同步说话）` |

## Step 6: 素材映射表
