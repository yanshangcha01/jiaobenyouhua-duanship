## Step 0：出镜人物设定（强制前置）

> **⚠️ 没做这一步就动笔，AI 生成的视频人物会"每镜换脸"。**
> 这是 v3.0 相对 v2.0 最关键的补丁——人物一致性是 AI 视频生成最大的坑。

### 0.1 确认是否有人物定妆照

```
□ 有现成定妆照
  → 直接使用，跳到 Step 1
  → 记录路径：/Volumes/外置硬盘/.../人物定妆照/XX店.png

□ 没有定妆照
  → 继续 0.2 创造人物档案
```

### 0.2 创造人物档案（7 个维度必填）

| 维度 | 必填项 | 填写示例 |
|------|--------|----------|
| **姓名** | 给角色起个名字（方便后续分镜引用） | 小宇 |
| **年龄 + 性别** | 准确到年龄段 | 男，25-28 岁 |
| **职业身份** | 一句话说清人设 | 兼职美食探店博主 |
| **性格特征** | 3-5 个关键词 | 亲和 / 幽默 / 实在 / 热情 |
| **说话风格** | 语速 + 口头禅 + 拉近关系的词 | 语速快节奏强，常用"家人们""我跟你说""真的绝了" |
| **穿搭设定** | 上装 / 下装 / 鞋 / 配饰 | 深灰圆领卫衣 / 黑色束脚裤 / 白色运动鞋 / 银色手链 |
| **气质关键词** | 5 个以内 | 亲和 / 幽默 / 实在 / 热情 / 接地气 |

### 0.3 生成人物 AI 绘图提示词库（6 种图必出）

> 把这套提示词丢进 Midjourney/即梦/SD，先把人物参考图生出来，后续所有分镜 AI 视频生成都用这套图作为角色一致性参考。

| 序号 | 图片类型 | 用途 | 是否必出 |
|------|---------|------|---------|
| 1 | **正面全身图** | 站姿全身，展示穿搭全貌 | ✅ 必出 |
| 2 | **正面半身图** | 胸部以上构图，胸部以上定妆参考 | ✅ 必出 |
| 3 | **侧面图** | 侧面轮廓，用于转身、行走场景 | ✅ 必出 |
| 4 | **背面图** | 背影，用于走进门店等场景 | ✅ 必出 |
| 5 | **面部特写** | 脸部细节，口型同步参考 | ✅ 必出 |
| 6 | **5 种表情参考图** | 竖大拇指 / 惊讶兴奋 / 皱眉舒展 / 真诚微笑 / 拍空位叫朋友 | ✅ 必出 |

[肤色]（如：皮肤健康小麦色），[五官]（如：五官立体偏瘦长脸），

## Step 1：台词深度分析

### 1.1 信息密度

- 这句话传递了什么核心卖点？
- 观众需要记住几个关键信息？
- 哪些词是钩子（吸引注意力）？

### 1.2 情绪曲线

- 这句话应该是什么语气？（兴奋/惊讶/真诚/俏皮/紧迫感）
- 前后句的情绪衔接逻辑是什么？
- 有没有情绪的递进或反转？

### 1.3 角色定位

- 谁是说话的人？（探店达人 / 店员 / 顾客视角）
- TA 和观众的关系是什么？（闺蜜分享 / 专业推荐 / 猎奇发现）
- 说话时的状态是什么？（刚发现宝藏 / 吃嗨了 / 准备安利）

### 1.4 动作匹配

- 说这句台词时，手在做什么？
- 身体朝向哪里？
- 眼神看向哪里？（镜头 / 菜品 / 环境）
- 有没有实物交互？（指着东西 / 拿着食物 / 夹起一块肉）

### 1.5 台词拆解输出格式

| 台词原文 | 核心信息 | 情绪基调 | 人物动作 |
|---------|---------|---------|---------|
| "别再纠结聚餐去哪了" | 抛出痛点 | 感同身受+轻松 | 双手摊开/拍大腿，眉头皱起后舒展 |
| ... | ... | ... | ... |

## Step 2：场景精准匹配

### 原则

**每句台词必须找到对应的场景支撑，不能凭空说。**

| 台词类型 | 应匹配场景 | 视觉优先级 |
|---------|-----------|----------|
| "藏在 XX 被我挖到了" | 门店外景 + 人物指路 | ⭐⭐⭐ |
| "各民族特色牛肉" | 民族系列菜品特写 | ⭐⭐⭐ |
| "连锁开很久，回头客多" | 店内人气画面 / 排队画面 / 大厅环境 | ⭐⭐ |
| "两位数的价格" | 小票展示 / 整桌菜品 | ⭐⭐⭐ |
| "好吃到爆" | 人物咬下食物的真实反应 | ⭐⭐⭐ |

### AI 生成场景指导

如果实拍素材不足，需要用 AI 绘图工具补充的场景：
- **门头环境图**：生成符合"张六娘"品牌色（红色招牌、民族风元素）的店铺外观
- **人物在场景中**：人物站在门口指门头、坐在桌前夹肉、举着饮料干杯
- **菜品氛围图**：带品牌标识的菜品陈列背景、热气腾腾的烤盘特写

## Step 3：分镜细节细化（节奏分配公式 + 细粒度连贯写法）

### 3.1 20 秒标准节奏公式（必背）

```
┌─────────────────────────────────────────────────────────────┐
│  0-3s   │  3-6s     │  6-10s    │  10-15s   │  15-20s      │
│  Hook   │  卖点1    │  卖点2    │  高潮     │  CTA         │
│  痛点/  │  解决     │  利益放大 │  体验     │  行动号召    │
│  惊喜   │  方案     │  价值     │  享受     │  收尾        │
└─────────────────────────────────────────────────────────────┘
```

**节奏分配原则**：
- **Hook（0-3s）**：必须 3 秒内抛出钩子，前 0.5 秒最关键
- **卖点1（3-6s）**：核心差异化卖点
- **卖点2（6-10s）**：第二卖点 / 价格冲击
- **高潮（10-15s）**：情绪爆发点，"好吃到爆"或产品特写轰炸
- **CTA（15-20s）**：行动号召 + 品牌信息

### 3.2 人物细节 5 维度（每个分镜必填）

1. **表情**：眼睛（睁大/眯眼/瞪圆）/ 嘴型（张大/抿嘴/微笑）/ 眉毛（上扬/紧皱）
2. **姿势**：站立 / 坐着 / 半蹲；身体朝向 / 重心分布
3. **手势**：手指指向 / 手掌张开 / 双手比划 / 手持物品
4. **眼神**：直视镜头 / 看向菜品 / 看向远处 / 上下扫视
5. **肢体动作**：拍手 / 比耶 / 竖大拇指 / 捂嘴 / 擦嘴角

### 3.3 逐分镜填写格式（细粒度连贯写法）

> **v3.0 强制要求**：用连贯叙事写每个分镜，不要再分块。
> 这样 AI 视频工具能直接理解整个画面的流动，而不是把表格字段割裂开。

每个分镜按以下顺序连贯填写：

```
### 分镜N｜时间：X-Y秒

**地点**：（具体拍摄位置）
**人物**：（同前/具体描述）
**动作**：（连贯描述整个分镜中人物在做什么）
**台词**："原话"
**情绪**：（情绪曲线变化）
**表情**：（表情的变化过程）

> **【AI 绘图提示词】**
> （一段连贯中文描述）

> **【AI 视频提示词】**
> （一段连贯中文描述，开头必须加 AI 负面提示词）
```

## Step 4：自检提问（7 问，每次必做）

> **v3.0 升级**：在 v2.0 的 6 问基础上，把 Q0 人物设定检查从附录提到最前面，作为第 0 问。

### Q0：有现成的人物设定吗？

```
□ 有现成定妆照
  → 记录路径，跳过 Step 0.2/0.3
  → 检查后续每个分镜的人物描述是否和定妆照一致

□ 没有定妆照
  → 必须先完成 Step 0.2 创造人物档案
  → 必须先完成 Step 0.3 生成 6 种人物参考图
  → 后续每个分镜的人物描述必须严格使用档案里的描述词
```

### Q1：这个脚本写的是什么人？
- [ ] 明确回答：这是一位什么样的博主（风格/气质/说话方式）
- [ ] 和 Step 0 的人物档案是否一致
- [ ] 跨分镜人物外貌/服装/配饰是否统一（**没有的话人物会"换脸"**）

### Q2：在哪里？
- [ ] 明确回答：具体是哪一家门店
- [ ] 场景识别度高不高？（门头、环境、品牌标识够不够清晰）
- [ ] 跨分镜场景是否一致（不要前面在宝龙店后面突然变成万达店）

### Q3：在干什么事？
- [ ] 明确回答：探店/吃播/推荐/种草？
- [ ] 动作逻辑合理吗？（先进店→先看到环境→先选菜→再吃→最后总结）

### Q4：观众能看懂吗？
- [ ] 核心信息有没有遗漏？（店名、位置、价格、卖点）
- [ ] 有没有太隐晦的画面？
- [ ] 每一句台词有没有对应的视觉支撑？

### Q5：这个视频的节奏对吗？
- [ ] 20 秒的时长是否合理分配？
- [ ] 前 3 秒有没有抓人的 hook？
- [ ] 高潮点是否在 10-15 秒区间？
- [ ] CTA 是否有清晰的行动指令？

### Q6：分镜提示词够不够具体？
- [ ] 每个分镜的画面描述能不能让 AI 工具（绘图+视频）直接理解？
- [ ] AI 提示词是否包含 8 要素（人物外貌/服装/姿势/表情/场景/光线/构图/台词）？
- [ ] AI 视频提示词是否包含**台词原文**（用于 AI 工具对口型同步）？

### Q7：AI 负面提示词加了吗？
- [ ] 脚本开头是否声明了 AI 负面提示词？
- [ ] **每一条** AI 绘图提示词开头是否都加了？
- [ ] **每一条** AI 视频提示词开头是否都加了？

## Step 5：输出分镜提示词（AI Image Prompt）

### 5.1 8 要素检查清单

每条 AI 绘图提示词必须包含以下 8 个要素（缺一个 AI 就会"自由发挥"）：

```
□ 人物外貌（年龄/性别/发型/肤色/五官）
□ 人物服装（上装/下装/鞋/配饰）
□ 姿势（站/坐/蹲 + 身体朝向）
□ 表情（眼睛/嘴型/眉毛）
□ 场景（地点/环境/品牌元素）
□ 光线（自然光/暖光/侧光/逆光）
□ 构图（9:16/16:9 + 景别 + 角度）
□ 风格调性（真实摄影/浅景深/不要卡通/不要插画）
```

### 5.2 格式模板

```
一位[年龄][性别]（如：25-28岁男性），[发型]，[肤色]，[五官]，
穿着[上装]、[下装]、[鞋]，[配饰]，
[姿势+表情]，
[景别/构图]，
[场景+光线]，
[风格调性]，
高清画质
```

### 5.3 提示词示例

```
一位 25-28 岁亚洲男性，黑色短发自然向上梳理，皮肤健康小麦色，五官立体偏瘦长脸，
穿着深灰色圆领宽松纯棉卫衣、黑色休闲束脚裤、白色运动鞋，
右手腕戴银色手链，
正对镜头直立站姿，双手自然垂放身体两侧，
9:16 竖版正面全身构图，
明亮现代餐厅内，浅景深虚化暖色调餐厅背景，
真实人类摄影照片质感，不要卡通不要插画，自然光照，高清画质
```

## Step 6：输出 AI 视频生成提示词（细粒度连贯写法）

### 6.1 强制 AI 负面提示词前缀

> **⚠️ 每一条 AI 视频提示词开头都必须加上这一段**：
>
> ```
> 不要字幕，不要背景音乐，人物要有真实感，保持真实皮肤的质感，食物类画面要贴近真实食物，自然状态下的纹理和质感。
> ```

### 6.2 细粒度连贯写法（v3.0 强制要求）

每条 AI 视频提示词按以下顺序连贯写完（不要分块）：

```
[AI 负面提示词] + @角色参考图 + 时间 + 地点 +
动作序列 + 台词（口型同步） + 话语对应的动作 +
表情变化 + 语气 + 情绪 + 镜头/光线/画面细节
```

> **⚠️ v3.0 关键优化（2026-07-15）**：
> **AI 视频提示词里不需要再重复人物外貌/服装/配饰**，因为 Step 0 已经生成了 6 种人物参考图，AI 视频工具（即梦/可灵/Kling）支持上传参考图 + @ 角色名。
> - ❌ 错误：还在写"22-25岁亚洲女性，粉色双麻花辫，皮肤白皙透亮，穿着浅彩色碎花休闲短款连衣裙，白色帆布鞋，脖颈戴简约银色项链……"
> - ✅ 正确：直接写 `@小桃 站在烤肉餐厅内……`，工具会自动从参考图里读出人物外貌/服装/配饰
> - 但**台词、动作、表情、语气、情绪、镜头**这些**必须**详细写（这些是参考图无法告诉 AI 的）

### 6.3 @ 角色参考图机制详解

```
┌─────────────────────────────────────────────────────┐
│ Step 0.3 生成的 6 种人物参考图                        │
│   ↓                                                  │
│ 上传到 AI 视频工具的"参考图"位置                      │
│   ↓                                                  │
│ 提示词里写 @小桃 / @角色名                            │
│   ↓                                                  │
│ AI 自动从参考图里读出：                               │
│   - 人物外貌（年龄/性别/发型/肤色/五官）               │
│   - 服装（上装/下装/鞋/配饰）                          │
│   - 气质/表情基线                                     │
└─────────────────────────────────────────────────────┘
```

**可以省略**（参考图已提供）：
- 人物年龄/性别
- 发型/发色/肤色/五官描述
- 服装（上装/下装/鞋）
- 配饰（项链/手链/手表）
- 身高/体型
- 气质关键词

**必须保留**（参考图给不出）：
- 动作序列（参考图不会动）
- 台词原文（口型同步必需）
- 话语对应的动作（分镜特异性）
- 表情变化（参考图是静态）
- 语气 / 情绪
- 镜头 / 运镜 / 光线
- 场景（参考图是人物图，不含背景）
