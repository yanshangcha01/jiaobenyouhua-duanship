# 抖音脚本创作方法论规范

## 第一步：台词深度分析

在写脚本之前，必须先对每句台词进行拆解，包括：

### 台词拆解维度

1. **信息密度**
   - 这句话传递了什么核心卖点？
   - 观众需要记住几个关键信息？
   - 哪些词是钩子（吸引注意力）？

2. **情绪曲线**
   - 这句话应该是什么语气？（兴奋/惊讶/真诚/俏皮/紧迫感）
   - 前后句的情绪衔接逻辑是什么？
   - 有没有情绪的递进或反转？

3. **角色定位**
   - 谁是说话的人？（探店达人 / 店员 / 顾客视角）
   - TA和观众的关系是什么？（闺蜜分享 / 专业推荐 / 猎奇发现）
   - 说话时的状态是什么？（刚发现宝藏 / 吃嗨了 / 准备安利）

4. **动作匹配**
   - 说这句台词时，手在做什么？
   - 身体朝向哪里？
   - 眼神看向哪里？（镜头 / 菜品 / 环境）
   - 有没有实物交互？（指着东西 / 拿着食物 / 夹起一块肉）

---

## 第二步：场景精准匹配

### 原则

**每句台词必须找到对应的场景支撑，不能凭空说。**

| 台词类型 | 应匹配场景 | 视觉优先级 |
|---------|-----------|----------|
| "藏在XX被我挖到了" | 门店外景 + 人物指路 | ⭐⭐⭐ |
| "各民族特色牛肉" | 民族系列菜品特写 | ⭐⭐⭐ |
| "连锁开很久，回头客多" | 店内人气画面 / 排队画面 / 大厅环境 | ⭐⭐ |
| "两位数的价格" | 小票展示 / 整桌菜品 | ⭐⭐⭐ |
| "好吃到爆" | 人物咬下食物的真实反应 | ⭐⭐⭐ |

### AI生成场景指导

如果实拍素材不足，需要用AI绘图工具补充的场景：
- **门头环境图**：生成符合品牌色（红色招牌、民族风元素）的店铺外观
- **人物在场景中**：人物站在门口指门头、坐在桌前夹肉、举着饮料干杯
- **菜品氛围图**：带品牌标识的菜品陈列背景、热气腾腾的烤盘特写

## 第三步：分镜细节细化

### 人物细节要求

每个分镜必须明确标注：

1. **表情**
   - 眼睛睁大（惊喜） / 眯眼笑（满意） / 挑眉（好奇）
   - 嘴型：张大（惊叹） / 抿嘴（品尝回味） / 微笑（友好）
   - 眉毛：上扬（兴奋） / 紧皱（疑惑）

2. **姿势**
   - 站立 / 坐着 / 半蹲（拍菜品低位角度）
   - 身体倾斜方向（面向镜头 / 侧身看菜品）
   - 重心分布（向前倾表示期待 / 后仰表示满足）

3. **手势**
   - 手指指向（具体目标：门头、菜品、价格）
   - 手掌张开（展示一整桌菜）
   - 双手比划（"两位数的两位数"比出数字）
   - 手持物品（举起饮料、夹起一片肉）

4. **眼神**
   - 直视镜头（和观众对话感）
   - 看向菜品（引导观众注意）
   - 看向远处门头（交代位置信息）
   - 上下扫视（展示丰富度）

5. **肢体动作**
   - 拍手（兴奋）
   - 比耶（开心）
   - 竖起大拇指（好评）
   - 捂嘴（被好吃到震惊）
   - 擦嘴角（吃饱满足感）

## 第四步：自检提问（每次必做）

在最终确认脚本前，必须回答以下问题：

### Q1：这个脚本写的是什么人？
- [ ] 明确回答：这是一位什么样的博主（风格/气质/说话方式）
- [ ] 和人物定妆照是否符合（同一角色贯穿全场）

### Q2：在哪里？
- [ ] 明确回答：具体是哪一家门店/地点
- [ ] 场景识别度高不高？（门头、环境、品牌标识够不够清晰）

### Q3：在干什么事？
- [ ] 明确回答：探店/吃播/推荐/种草？
- [ ] 动作逻辑合理吗？（先进店→先看到环境→先选菜→再吃→最后总结）

### Q4：观众能看懂吗？
- [ ] 核心信息有没有遗漏？（店名、位置、价格、卖点）
- [ ] 有没有太隐晦的画面？
- [ ] 每一句台词有没有对应的视觉支撑？

### Q5：这个视频的节奏对吗？
- [ ] 20秒的时长是否合理分配？
- [ ] 前3秒有没有抓人的hook？
- [ ] 有没有拖沓或赶进度的感觉？

### Q6：分镜提示词够不够具体？
- [ ] 每个分镜的画面描述能不能让AI工具（绘图+视频）直接理解？
- [ ] 有没有包含：人物（外貌/衣服/姿势）、场景（室内/室外/光线）、物体（菜品/道具）、情绪（表情）

## 第五步：输出分镜提示词（AI Image Prompt）

每个分镜都需要生成对应的 **AI绘图提示词**，用于Midjourney/Stable Diffusion/即梦生图工具生成分镜图。

### 提示词格式模板

```
【画面构图】：（景别+角度+构图方式）
【人物描述】：（外貌+服装+姿势+表情+眼神+手势）
【场景环境】：（地点+光线+氛围+品牌元素）
【道具/菜品】：（具体出现的物品，摆放方式）
【风格调性】：（色调+质感+参考风格）
```

### 示例

```
画面构图：竖版9:16中景，人物位于画面左侧1/3处，右侧留白展示门头招牌
人物描述：年轻女性探店博主，长发微卷，穿浅色休闲连衣裙，右手食指指向身后门头，左手自然叉腰，表情兴奋（嘴巴微张+眼睛睁大），目光直视镜头
场景环境：商场内部，暖色调灯光，背景为"张六娘"红色招牌，招牌下方有民族风装饰海报
道具/菜品：无明显道具，重点突出人物和招牌的互动关系
风格调性：明亮清新，抖音探店风格，高饱和度，商业摄影质感
```

## 第六步：输出AI视频生成提示词

每个分镜还需要生成对应的 **AI视频生成提示词**，用于即梦视频/可灵/Kling/Runway/Sora等工具生成分镜视频。

### 视频提示词格式模板（细粒度）

```
【景别】：（近景/远景/全景/特写/超特写 + 俯拍/仰拍/平拍/正面/侧面）
【运镜】：（固定镜头/缓慢推进/拉远/跟拍/横移/环绕/快切）
【时长】：几秒～几秒钟
【人物位置】：人物在哪里（场景描述）
【动作序列】：按时间顺序列出人物做的一系列动作
【台词】：这句分镜的台词原文
【话语对应的动作】：说到每句话时人物在做什么动作（逐句对应）
【表情】：整个分镜的表情变化（阶段描述）
【说话的语气】：语速/音调/重音/情感色彩
【人物情绪】：整体情绪状态
【画面细节】：光线/背景/道具/氛围等
【视频提示词】：（一段可以直接复制进AI视频工具的完整中文描述，必须包含台词原文，AI视频工具会根据台词同步人物口型和动作）
```

## 第七步：脚本完整性检查清单

最终交付的脚本必须包含以下模块：

- [x] **台词分析段落** — 拆解每句话的信息/情绪/动作
- [x] **分镜脚本表格** — 序号 | 时长 | 景别 | 台词 | 画面 | 表情动作 | 人物定妆照 | 图片素材
- [x] **逐镜AI绘图提示词** — 每个分镜都有独立的Midjourney/Stable Diffusion提示词
- [x] **逐镜AI视频提示词** — 每个分镜都有独立的细粒度视频提示词（含景别+运镜+人物动作+台词对应动作+表情+语气+情绪），可直接复制进即梦/可灵/Kling
- [x] **图片素材映射表** — 每条分镜都关联了实拍素材或AI生成方案
- [x] **自检问答记录** — 6个问题全部回答完毕
- [x] **视频参数标注** — 时长/比例/BGM/平台
